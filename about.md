I'm Anenth — a Staff Front-End Engineer with over a decade of experience building products for the web.

My journey began with CSS and the sheer magic of making things come alive in the browser. That early fascination evolved into a deep passion for front-end engineering — from crafting pixel-perfect interfaces to architecting scalable, performant web applications.

## What I do

I specialize in building exceptional user experiences with modern web technologies. My focus areas include:

- **Performance Engineering** — Core Web Vitals optimization, rendering performance, and bundle analysis
- **Design Systems** — Building and scaling component libraries that teams love to use
- **Modern JavaScript/TypeScript** — React, Next.js, and the broader ecosystem
- **Web Platform APIs** — Leveraging the latest browser capabilities for better UX

## Philosophy

I believe great software is built at the intersection of engineering excellence and developer empathy. Code should be readable, maintainable, and a joy to work with. I care deeply about the developer experience — both for the teams I work with and for the end users we serve.
